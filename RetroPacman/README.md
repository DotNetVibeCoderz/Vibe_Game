# RetroPacman

A classic arcade game clone built with .NET 8 and Avalonia UI.

## Features
- **Classic Gameplay**: Eat pellets, avoid ghosts.
- **Power Pellets (Coins)**: Eat the large blinking coins to turn the tables! Ghosts turn blue and can be eaten for extra points.
- **Improved Graphics**: Animated Pacman mouth, Ghosts with eyes that look around.
- **Cross-Platform**: Runs on Windows, Linux, and macOS.
- **Progressive Difficulty**: Faster ghosts and smarter AI as levels increase.

## Controls
- **Arrow Keys**: Move Pacman
- **ESC**: Pause / Menu
- **N**: New Game (in Menu)
- **Q**: Quit

## How to Run
1. Install .NET 8 SDK.
2. Navigate to the project folder.
3. Run: `dotnet run`

Enjoy!
