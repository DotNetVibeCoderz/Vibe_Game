# RubikCube3D

![License](https://img.shields.io/badge/license-MIT-blue.svg)

**English**

RubikCube3D is a cross-platform 3D Rubik's Cube simulator built with C# and Avalonia UI. It features a realistic 3D interactive cube that you can rotate, scramble, and solve.

## Features
- **3D Interaction**: Rotate the entire cube view using the mouse (Left Click + Drag).
- **Cube Control**: Rotate individual faces using keyboard shortcuts (F, B, U, D, L, R). Hold SHIFT for counter-clockwise rotations.
- **Dimensions**: Supports 2x2, 3x3, and 4x4 cubes.
- **Solver**: Automatic "Solve" button (reverses moves).
- **Undo/Redo**: Step through your history.
- **Cross-Platform**: Runs on Windows, Linux, and macOS.

## Controls
- **Mouse**: Left-click and drag to rotate the camera view.
- **Scroll Wheel**: Zoom in/out.
- **Keyboard**:
  - `F`: Front Face
  - `B`: Back Face
  - `U`: Up Face
  - `D`: Down Face
  - `L`: Left Face
  - `R`: Right Face
  - `Shift + Key`: Rotate Counter-Clockwise
  - `Ctrl + Z`: Undo
  - `Ctrl + Y`: Redo
  - `F5`: Scramble

## Installation
1. Ensure .NET 8.0 SDK is installed.
2. Clone the repository.
3. Run `dotnet run` in the project directory.

**Bahasa Indonesia**

RubikCube3D adalah simulator Rubik 3D lintas platform yang dibuat menggunakan C# dan Avalonia UI. Aplikasi ini menampilkan kubus 3D interaktif yang realistis yang dapat Anda putar, acak, dan selesaikan.

## Fitur
- **Interaksi 3D**: Putar tampilan kubus menggunakan mouse (Klik Kiri + Geser).
- **Kontrol Kubus**: Putar sisi individu menggunakan pintasan keyboard (F, B, U, D, L, R). Tahan SHIFT untuk putaran berlawanan arah jarum jam.
- **Dimensi**: Mendukung kubus 2x2, 3x3, dan 4x4.
- **Penyelesai**: Tombol "Solve" otomatis (membalikkan langkah).
- **Undo/Redo**: Telusuri riwayat langkah Anda.
- **Lintas Platform**: Berjalan di Windows, Linux, dan macOS.

## Kontrol
- **Mouse**: Klik kiri dan geser untuk memutar pandangan kamera.
- **Roda Mouse**: Perbesar/Perkecil (Zoom).
- **Keyboard**:
  - `F`: Sisi Depan
  - `B`: Sisi Belakang
  - `U`: Sisi Atas
  - `D`: Sisi Bawah
  - `L`: Sisi Kiri
  - `R`: Sisi Kanan
  - `Shift + Tombol`: Putar Berlawanan Arah Jarum Jam
  - `Ctrl + Z`: Undo (Kembali)
  - `Ctrl + Y`: Redo (Ulangi)
  - `F5`: Acak (Scramble)

## Instalasi
1. Pastikan .NET 8.0 SDK sudah terinstal.
2. Clone repositori ini.
3. Jalankan `dotnet run` di direktori proyek.
