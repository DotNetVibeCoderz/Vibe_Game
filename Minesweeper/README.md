### About the Project
This is a classic **Minesweeper** game recreated using **C#** and **Windows Forms (.NET 9.0)**. It brings back the nostalgia of the old Windows era with a modern codebase. Use logic to uncover safe squares and flag the mines!

### Features
*   **3 Difficulty Levels:**
    *   Beginner (9x9 grid, 10 mines)
    *   Intermediate (16x16 grid, 40 mines)
    *   Expert (30x16 grid, 99 mines)
*   **Classic Mechanics:**
    *   **First Click Safe:** You will never hit a mine on your first click.
    *   **Flood Fill:** Automatically reveals empty areas.
    *   **Number Coloring:** Classic color coding for numbers (1=Blue, 2=Green, 3=Red, etc.).
*   **Interactive UI:**
    *   **Smiley Face Button:** Reacts to gameplay (ğŸ™‚ Normal, ğŸ˜® Clicking, ğŸ˜ Win, ğŸ˜µ Loss). Click it to restart!
    *   **Timer & Mine Counter:** Keeps track of your time and remaining flags.
    *   **Right-Click Flagging:** Mark suspected mines with a flag (ğŸš©).

### Controls
*   **Left Click:** Reveal a cell.
*   **Right Click:** Place/Remove a flag (ğŸš©) on a cell.
*   **Smiley Button:** Restart the game.
*   **Menu:** Change difficulty or exit via the top menu.

### How to Run
1.  Ensure you have **.NET 9.0 SDK** installed.
2.  Open your terminal/command prompt.
3.  Navigate to the project directory.
4.  Run the following command:
    ```bash
    dotnet run
    ```
