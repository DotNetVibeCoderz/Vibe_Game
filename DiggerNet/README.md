# DiggerNet

A clone of the classic arcade game "Digger", built with C# and Avalonia UI.

## Features
- **Core Gameplay**: Dig tunnels, collect emeralds, push gold bags, and avoid Nobbin and Hobbin.
- **Audio**: Retro sound effects (digging, shooting, collecting, etc.).
- **Menus**: Complete menu system with Start, Options, About, and Exit.
- **Responsive**: Dynamic resizing and scaling.

## Controls
- **Arrow Keys**: Move Digger
- **F1**: Fire Weapon
- **Space**: Pause / Restart Game
- **Escape**: Return to Menu
- **+ / -**: Adjust Game Speed

## Requirements
- .NET 8.0 SDK

## How to Run
1. Navigate to the project folder.
2. Run `dotnet run`.
