# Particle Combat

**Particle Combat** is a fast-paced survival shooter game built with C# and MonoGame. 
Created by **Jacky the Code Bender** from **Gravicode Studios**.

## Features
- **Survival Shooter Gameplay**: Infinite waves of enemies.
- **Particle Effects**: Colorful explosions and visual chaos.
- **Simple Controls**: Easy to pick up and play.
- **Highscore System**: Compete with yourself for the best score.

## Controls
- **W, A, S, D**: Move Player
- **Mouse**: Aim
- **Left Click**: Shoot
- **ESC**: Pause / Menu
- **Enter**: Select Menu Option / Resume

## How to Run
1. Ensure you have .NET 6.0 or later installed.
2. Open the project folder.
3. Run `dotnet run` in the terminal.
