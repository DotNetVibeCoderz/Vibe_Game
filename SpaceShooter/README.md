# Space Shooter - Jacky The Code Bender

## Deskripsi (Bahasa Indonesia)
Ini adalah game arcade tembak-tembakan luar angkasa klasik yang dikembangkan menggunakan C# WinForms.
Fitur Utama:
- **Multi Level**: Kesulitan meningkat setiap naik level.
- **Boss Fight**: Musuh besar muncul di akhir setiap level.
- **Berbagai Jenis Musuh**: Scout (cepat), Fighter (zig-zag), dan Boss.
- **Latar Belakang Parallax**: Bintang-bintang yang bergerak memberikan efek kedalaman.
- **Kontrol Halus**: Pergerakan player yang responsif.
- **Menu Game**: Tekan ESC untuk kembali ke menu utama.

**Kontrol:**
- Panah / WASD: Bergerak
- Spasi / K: Menembak
- ESC: Menu / Kembali

## Description (English)
A classic space shooter arcade game developed using C# WinForms.
Key Features:
- **Multi Level**: Higher difficulty as you level up.
- **Boss Fight**: A giant enemy appears at the end of each level.
- **Various Enemy Types**: Scout (fast), Fighter (zig-zag), and Boss.
- **Parallax Background**: Moving stars for depth effect.
- **Smooth Controls**: Responsive player movement.
- **Game Menu**: Press ESC to return to the main menu.

**Controls:**
- Arrows / WASD: Move
- Space / K: Shoot
- ESC: Menu / Back
