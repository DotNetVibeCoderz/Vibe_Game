# RollerBall - Zuma Clone

RollerBall is a puzzle arcade game inspired by Zuma, built with C# and Avalonia UI.

## Features
- **Marble Shooting Mechanic**: Control the stone frog idol to shoot colored balls.
- **Path Progression**: Balls roll along a spiral track towards the center skull.
- **Match-3 Gameplay**: Match 3 or more balls of the same color to clear them.
- **Level System**: Progression with increasing difficulty.
- **Immersive Visuals**: Aztec/Mayan themed background and effects.

## Controls
- **Mouse Move**: Aim the shooter.
- **Left Click**: Shoot ball.
- **Escape / Menu Button**: Pause game.

## How to Run
1. Ensure you have the .NET 8 SDK installed.
2. Navigate to the project folder.
3. Run the following command:
   ```bash
   dotnet run
   ```

## Requirements
- .NET 8.0 or later
- Windows, Linux, or macOS (Cross-platform support via Avalonia)

# RollerBall - Klon Zuma

RollerBall adalah game arcade teka-teki yang terinspirasi oleh Zuma, dibuat dengan C# dan Avalonia UI.

## Fitur
- **Mekanisme Menembak Kelereng**: Kendalikan patung kodok batu untuk menembak bola berwarna.
- **Progres Jalur**: Bola menggelinding di sepanjang jalur spiral menuju tengkorak tengah.
- **Gameplay Cocokkan-3**: Cocokkan 3 bola atau lebih dengan warna yang sama untuk menghapusnya.
- **Sistem Level**: Progres dengan tingkat kesulitan yang meningkat.
- **Visual Imersif**: Latar belakang dan efek bertema Aztec/Maya.

## Kontrol
- **Gerakan Mouse**: Arahkan penembak.
- **Klik Kiri**: Tembak bola.
- **Escape / Tombol Menu**: Jeda permainan.

## Cara Menjalankan
1. Pastikan Anda telah menginstal .NET 8 SDK.
2. Buka folder proyek.
3. Jalankan perintah berikut:
   ```bash
   dotnet run
   ```

## Persyaratan
- .NET 8.0 atau lebih baru
- Windows, Linux, atau macOS (Dukungan lintas platform melalui Avalonia)
