# CrossyGame

![Icon](https://img.icons8.com/color/48/000000/chicken.png)

## English

### Description
**CrossyGame** is an endless arcade hopper game inspired by the classic "Crossy Road". Built using **C#** and **Avalonia UI**, this game runs smoothly on Windows, Linux, and macOS. The goal is simple: hop as far as you can across busy roads, treacherous rivers, and dangerous train tracks without getting hit or drowning!

### Features
- **Endless Gameplay**: The world is generated procedurally as you move forward.
- **Dynamic Obstacles**:
  - **Roads**: Avoid speeding cars.
  - **Rivers**: Hop on floating logs to cross safely.
  - **Rail Tracks**: Watch out for the warning lights before the express train zooms by!
- **High Score System**: Track your progress based on the distance traveled.
- **Cross-Platform**: Runs natively on major desktop operating systems via Avalonia.

### How to Run
1. Ensure you have the **.NET 8.0 SDK** installed.
2. Open a terminal in the project directory.
3. Run the following command:
   ```bash
   dotnet run
   ```

### Controls
- **Move Forward**: `Up Arrow` or `W`
- **Move Backward**: `Down Arrow` or `S`
- **Move Left**: `Left Arrow` or `A`
- **Move Right**: `Right Arrow` or `D`
- **Restart Game**: `R` (When Game Over screen appears)
