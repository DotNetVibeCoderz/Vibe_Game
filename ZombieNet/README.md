# ZombieNet - Tower Defense Game

## Description (English)
ZombieNet is a thrilling Tower Defense game built with C# and MonoGame. Defend your base from waves of relentless zombies by strategically placing towers along their path. Manage your resources, upgrade your defenses, and survive as long as possible!

### Features
- **Wave System**: Infinite waves of zombies with increasing difficulty.
- **Towers**: Place Gun Turrets to shoot enemies.
- **Economy**: Earn gold by killing zombies to buy more towers.
- **Lives**: Prevent zombies from reaching your base.
- **Visuals**: Retro pixel-art style graphics.

### Controls
- **Mouse Click**: Place a Tower on Grass tiles (Cost: 50 Gold).
- **Esc**: Exit the game.

### How to Run
1. Ensure you have .NET 8.0 SDK installed.
2. Navigate to the project folder.
3. Run the command: `dotnet run`
