# RetroBilliard - Retro 3D Billiard Game Project

**RetroBilliard** is a retro-style 3D billiard game built using C# and the Raylib-cs library. This game offers a simple yet addictive billiard experience with minimalist graphics and solid physics systems.

Created with love by **Jacky the Code Bender** from the **Gravicode Studios** team.

## Key Features
- **Game Modes:** Choose between Player vs Player (PvP) or face the AI in Player vs Computer (PvC).
- **Retro 3D Graphics:** Classic aesthetic visuals for the table and balls.
- **Physics System:** Realistic simulation of ball collisions, table friction, and wall bounces.
- **Orbit Camera:** Full camera control to view the table from various angles.
- **Interactive UI:** Complete main menu with easy navigation.

## Game Controls
- **WASD:** Move camera (Rotation and Zoom).
- **Left/Right Arrows:** Adjust cue aiming angle.
- **Space (Hold):** Set hit strength (Power Bar).
- **R:** Reset the game.
- **ESC:** Return to the main menu.

---

## Teknologi yang Digunakan / Technologies Used
- **Language:** C#
- **Graphics Library:** [Raylib-cs](https://github.com/ChrisDill/Raylib-cs)
- **Framework:** .NET 10.0

## Cara Menjalankan / How to Run
1. Pastikan Anda sudah menginstal .NET SDK (versi 10.0 atau yang kompatibel).
2. Clone atau download project ini.
3. Buka terminal di folder project.
4. Jalankan perintah:
   ```bash
   dotnet run
   ```
