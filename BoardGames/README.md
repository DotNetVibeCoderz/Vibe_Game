# Board Games Collection

**English**
This is a cross-platform Board Game collection built with C# and Avalonia UI. It features three classic games:
1. **Othello (Reversi)** - Strategy board game for two players.
2. **Checkers** - Classic draughts game.
3. **Tic Tac Toe** - Simple strategy game.

**Features:**
- Cross-Platform (Windows, Linux, macOS).
- Beautiful Dark Mode UI with Acrylic effects.
- AI Opponent with 3 Difficulty Levels (Beginner, Intermediate, Expert).
- Two Player Mode.
- Scoreboard and Timer.

**How to Run:**
1. Ensure .NET 8 SDK is installed.
2. Navigate to the project folder.
3. Run `dotnet run`.
