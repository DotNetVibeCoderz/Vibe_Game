# SudokuNet

A Sudoku game built with C# and Avalonia UI.

## Features
- 9x9 Grid Logic Puzzle
- Multiple Difficulty Levels: Easy, Medium, Hard, Expert, Extreme
- Mistake Tracking
- Timer
- Clean and modern interface (Fluent Theme)

## How to Run
1. Ensure you have .NET SDK installed.
2. Run `dotnet run` in the project directory.

## Controls
- Select a cell by clicking on it.
- Use the on-screen number pad to input numbers.
- Press 'C' to clear a cell.
