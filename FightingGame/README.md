# FightingGame - Jacky Edition

A 2D Fighting Game built with C# and Avalonia UI.

## Description
This is a one-on-one fighting game where you battle against an AI opponent. Use your martial arts skills to defeat the enemy before the time runs out!

## Controls
- **Arrow Keys**: Move Left/Right, Jump (Up)
- **Z**: Punch
- **X**: Kick
- **C**: Block

## Features
- Player vs AI
- Health Bars & Timer
- Simple Physics & Collision Detection
- 2D Pixel Art Graphics
- **Procedural Sound Effects**: Sounds are generated in real-time using math (no external files needed)!

## How to Run
1. Open the project in Visual Studio or VS Code.
2. Build and Run using `.NET 8.0`.
3. Or run `dotnet run` in the terminal.

# FightingGame - Edisi Jacky

Game pertarungan 2D yang dibuat dengan C# dan Avalonia UI.

## Deskripsi
Ini adalah game pertarungan satu lawan satu di mana Anda bertarung melawan lawan AI. Gunakan kemampuan bela diri Anda untuk mengalahkan musuh sebelum waktu habis!

- **Tombol Panah**: Bergerak Kiri/Kanan, Lompat (Atas)

## Fitur
- Pemain vs AI
- Bar Kesehatan & Timer
- Fisika Sederhana & Deteksi Tabrakan
- Grafik Pixel Art 2D
- **Efek Suara Prosedural**: Suara dihasilkan secara real-time menggunakan matematika (tidak perlu file eksternal)!

## Cara Menjalankan
1. Buka proyek di Visual Studio atau VS Code.
