**Tamagochi** adalah aplikasi simulasi hewan peliharaan digital (virtual pet) bergaya pixel art yang lucu dan interaktif. Dibuat menggunakan C# dan Windows Forms.

### Fitur Utama:
1. **Gaya Pixel Art**: Tampilan visual retro yang digambar secara prosedural.
2. **Siklus Hidup**: Pet berkembang dari Telur -> Bayi -> Dewasa. Jika tidak dirawat, pet bisa Mati.
3. **Penyimpanan Otomatis**: Kondisi pet akan tersimpan saat aplikasi ditutup dan dilanjutkan saat dibuka kembali.
4. **Interaksi**:
   - **Feed (Makan)**: Mengurangi lapar.
   - **Clean (Bersihkan)**: Menjaga kebersihan.
   - **Play (Main)**: Meningkatkan kebahagiaan.
   - **Sleep (Tidur)**: Memulihkan energi.
   - **Heal (Obat)**: Menyembuhkan saat sakit.
5. **Indikator**: Memantau Lapar, Kebersihan, Kebahagiaan, Energi, dan Kesehatan.
6. **Sistem Emosi**: Pet akan menunjukkan ekspresi (Senang, Marah, Sedih, Sakit) sesuai kondisinya.
7. **Pilihan Pet**: Tersedia Dino, Kucing, dan Robot.

### Cara Menjalankan:
1. Pastikan .NET SDK terinstall.
2. Buka folder project.
3. Jalankan perintah `dotnet run` atau buka file `.sln` dengan Visual Studio dan klik Start.

**Tamagochi** is a cute and interactive pixel-art style digital pet simulator. Built with C# and Windows Forms.

### Key Features:
1. **Pixel Art Style**: Procedurally drawn retro visuals.
2. **Lifecycle**: Pet evolves from Egg -> Baby -> Adult. If neglected, the pet can Die.
3. **Auto Save**: Pet state is saved when the app closes and restored upon reopening.
4. **Interactions**:
   - **Feed**: Reduces hunger.
   - **Clean**: Maintains hygiene.
   - **Play**: Increases happiness.
   - **Sleep**: Restores energy.
   - **Heal**: Cures sickness.
5. **Indicators**: Monitor Hunger, Hygiene, Happiness, Energy, and Health.
6. **Emotion System**: Pet shows expressions (Happy, Angry, Sad, Sick) based on its status.
7. **Pet Types**: Choose between Dino, Cat, and Robot.

### How to Run:
1. Ensure .NET SDK is installed.
2. Open the project folder.
3. Run `dotnet run` or open the `.sln` file in Visual Studio and click Start.
