# WarShipNet - 1945 Space Shooter

## English
WarShipNet is a vertical scrolling space shooter game inspired by the classic 1945 arcade games. Built with Blazor WebAssembly and Phaser 3.

### Features
- **Classic Arcade Action:** Dodge bullets and destroy enemies.
- **Multiple Enemies:** Fight against different types of ships and powerful bosses.
- **Difficulty Levels:** Choose from Beginner, Medium, or Hard.
- **Sound Effects:** Retro-style synthesized sound effects.
- **Cross-Platform:** Runs in any modern web browser.

### How to Play
1.  **Move:** Use Arrow Keys or WASD.
2.  **Shoot:** Press Spacebar.
3.  **Objective:** Survive as long as possible and defeat the bosses.

### Installation
1.  Open the project in Visual Studio or VS Code.
2.  Run the application (`dotnet run` or F5).
3.  Enjoy!
