# CrossWordsNet

CrossWordsNet is a modern, cross-platform Crossword Puzzle game built with C# and Avalonia UI. It features daily puzzles, random level generation, and a clean, dark-themed interface.

## Features
- **Random Level Generation**: Every game is unique!
- **Clean UI**: Minimalist design with dark mode support.
- **Timer & Score**: Track your solving speed and accuracy.
- **Hints**: Get help when you're stuck.
- **Keyboard Navigation**: Use arrow keys to navigate the grid easily.

## How to Play
1. **Launch the Game**: Run the executable.
2. **New Game**: Click "New Game" to start.
3. **Solve**: Click on a cell or use arrow keys to navigate. Type letters to fill in words.
4. **Clues**: Read Across and Down clues on the right panel.
5. **Check**: Click "Check" to verify your answers.
6. **Hint**: Click "Hint" to reveal the current cell (costs score).

## Requirements
- .NET 8.0 SDK or later

## Build and Run
```bash
dotnet restore
dotnet run
```

# CrossWordsNet (Bahasa Indonesia)

CrossWordsNet adalah permainan Teka-Teki Silang modern lintas platform yang dibangun dengan C# dan Avalonia UI. Game ini memiliki fitur teka-teki harian, pembuatan level acak, dan antarmuka bertema gelap yang bersih.

## Fitur
- **Pembuatan Level Acak**: Setiap permainan unik!
- **Antarmuka Bersih**: Desain minimalis dengan dukungan mode gelap.
- **Pengatur Waktu & Skor**: Lacak kecepatan dan akurasi penyelesaian Anda.
- **Petunjuk**: Dapatkan bantuan saat Anda buntu.
- **Navigasi Keyboard**: Gunakan tombol panah untuk menavigasi kisi dengan mudah.

## Cara Bermain
1. **Jalankan Game**: Buka aplikasi.
2. **Permainan Baru**: Klik "New Game" untuk memulai.
3. **Selesaikan**: Klik pada sel atau gunakan tombol panah untuk menavigasi. Ketik huruf untuk mengisi kata.
4. **Petunjuk**: Baca petunjuk Mendatar (Across) dan Menurun (Down) di panel kanan.
5. **Periksa**: Klik "Check" untuk memverifikasi jawaban Anda.
6. **Bantuan**: Klik "Hint" untuk membuka huruf pada sel saat ini (mengurangi skor).

## Persyaratan
- .NET 8.0 SDK atau yang lebih baru

## Cara Membangun dan Menjalankan
```bash
dotnet restore
dotnet run
```
