# LiteGame2D Collection / Koleksi LiteGame2D

**[Bahasa Indonesia]**

## Deskripsi
Ini adalah koleksi 6 mini-game klasik yang dibuat menggunakan C# dan Windows Forms. Proyek ini mendemonstrasikan logika game sederhana dan grafis 2D dasar (GDI+).

## Cara Menjalankan
1. Buka solusi `LiteGame2D.sln` dengan Visual Studio atau editor yang mendukung .NET.
2. Build dan Run project.
3. Gunakan tombol keyboard **Angka 1 sampai 6** untuk memilih game.

## Daftar Game & Kontrol

### 1. Snake Game (Ular)
Arahkan ular untuk memakan makanan merah. Jangan menabrak tembok atau ekor sendiri.
*   **Kontrol**: Tombol Panah (Atas, Bawah, Kiri, Kanan).
*   **Ganti Game**: Tekan `1`.

### 2. Flappy Bird
Terbangkan burung melewati pipa-pipa.
*   **Kontrol**: `Spasi` untuk lompat/terbang.
*   **Ganti Game**: Tekan `2`.

### 3. BomberMan
Letakkan bom untuk menghancurkan tembok dan musuh. Hati-hati terkena ledakan sendiri!
*   **Kontrol**:
    *   Tombol Panah: Bergerak.
    *   `Spasi`: Taruh Bom.
*   **Ganti Game**: Tekan `3`.

### 4. Tetris
Susun balok agar membentuk garis penuh untuk menghilangkannya.
*   **Kontrol**:
    *   Panah Kiri/Kanan: Geser balok.
    *   Panah Atas: Putar balok.
    *   Panah Bawah: Percepat turun.
*   **Ganti Game**: Tekan `4`.

### 5. Pinball
Pantulkan bola agar tidak jatuh ke bawah dan kumpulkan koin.
*   **Kontrol**:
    *   `Panah Kiri`: Flipper Kiri.
    *   `Panah Kanan`: Flipper Kanan.
    *   `Spasi`: Luncurkan bola (Plunger).
*   **Ganti Game**: Tekan `5`.

### 6. T-Rex Runner
Lompat menghindari kaktus.
*   **Kontrol**: `Spasi` untuk lompat.
*   **Ganti Game**: Tekan `6`.

**[English]**

## Description
This is a collection of 6 classic mini-games built using C# and Windows Forms. This project demonstrates simple game logic and basic 2D graphics (GDI+).

## How to Run
1. Open the `LiteGame2D.sln` solution with Visual Studio or a .NET compatible editor.
2. Build and Run the project.
3. Use keyboard number keys **1 to 6** to switch between games.

## Game List & Controls

### 1. Snake Game
Guide the snake to eat the red food. Avoid hitting walls or your own tail.
*   **Controls**: Arrow Keys (Up, Down, Left, Right).
*   **Switch Game**: Press `1`.

### 2. Flappy Bird
Fly the bird through the pipes.
*   **Controls**: `Space` to jump/fly.
*   **Switch Game**: Press `2`.

### 3. BomberMan
Place bombs to destroy walls and enemies. Watch out for your own explosions!
*   **Controls**:
    *   Arrow Keys: Move.
    *   `Space`: Place Bomb.
*   **Switch Game**: Press `3`.

### 4. Tetris
Arrange blocks to form full lines to clear them.
*   **Controls**:
    *   Left/Right Arrow: Move block.
    *   Up Arrow: Rotate block.
    *   Down Arrow: Speed up drop.
*   **Switch Game**: Press `4`.

### 5. Pinball
Keep the ball from falling and collect coins.
*   **Controls**:
    *   `Left Arrow`: Left Flipper.
    *   `Right Arrow`: Right Flipper.
    *   `Space`: Launch ball (Plunger).
*   **Switch Game**: Press `5`.

### 6. T-Rex Runner
Jump over obstacles (cacti).
*   **Controls**: `Space` to jump.
*   **Switch Game**: Press `6`.
