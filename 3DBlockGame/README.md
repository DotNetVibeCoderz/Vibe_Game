# 3DBlockGame (Minecraft Clone)

A simple Minecraft clone built with **Blazor WebAssembly** and **Three.js**.

## Features
- **Procedural Terrain**: Generates a simple world with grass, dirt, stone, and trees.
- **Building System**: Place and break blocks using Raycasting.
- **Physics**: Basic collision detection and jumping.
- **Textures**: Procedurally generated textures using HTML5 Canvas (no external assets needed).
- **Sound**: Synthesized sound effects using Web Audio API.

## Controls
- **W, A, S, D**: Move
- **SPACE**: Jump
- **Mouse**: Look around
- **Left Click**: Break block
- **Right Click**: Place block
- **1-5 Keys**: Select block type (Grass, Dirt, Stone, Wood, Leaves)

## How to Run
1. Ensure you have .NET 6.0 SDK or later installed.
2. Open the project folder in your terminal.
3. Run the command:
   ```bash
   dotnet watch run
   ```
4. Open your browser at the displayed URL (usually `https://localhost:7183` or `http://localhost:5000`).

# 3DBlockGame (Klon Minecraft)

Sebuah klon Minecraft sederhana yang dibuat menggunakan **Blazor WebAssembly** dan **Three.js**.

## Fitur
- **Terrain Prosedural**: Menghasilkan dunia sederhana dengan rumput, tanah, batu, dan pohon.
- **Sistem Bangunan**: Menaruh dan menghancurkan blok menggunakan Raycasting.
- **Fisika**: Deteksi tabrakan dasar dan melompat.
- **Tekstur**: Tekstur yang dihasilkan secara prosedural menggunakan HTML5 Canvas (tidak memerlukan aset eksternal).
- **Suara**: Efek suara sintesis menggunakan Web Audio API.

## Kontrol
- **W, A, S, D**: Bergerak
- **SPASI**: Lompat
- **Mouse**: Melihat sekeliling
- **Klik Kiri**: Hancurkan blok
- **Klik Kanan**: Taruh blok
- **Tombol 1-5**: Pilih jenis blok (Rumput, Tanah, Batu, Kayu, Daun)

## Cara Menjalankan
1. Pastikan Anda telah menginstal .NET 6.0 SDK atau yang lebih baru.
2. Buka folder proyek di terminal Anda.
3. Jalankan perintah:
   ```bash
   dotnet watch run
   ```
4. Buka browser Anda di URL yang ditampilkan (biasanya `https://localhost:7183` atau `http://localhost:5000`).
