# Maze 3D - Jacky's Edition

## English
**Maze 3D** is a retro-style maze game inspired by classic raycasting games like Wolfenstein 3D. Created entirely in C# without external game engines, this project demonstrates raw programming power.

### Features
*   **3D Raycasting Engine**: Built from scratch using C# and GDI+.
*   **Procedural Generation**: Every level has a unique maze layout.
*   **Visuals**: Custom generated textures for walls, doors, and treasures.
*   **Progressive Difficulty**: Levels get harder with more treasures to collect.
*   **No Assets Required**: All graphics and sounds are generated at runtime!

### How to Play
1.  Run the application.
2.  Use **W, A, S, D** or **Arrow Keys** to move and rotate.
3.  Collect all **Treasures** (Gold Cups) to score points.
4.  Find the **EXIT Door** (marked "EXIT") to proceed to the next level.
5.  Press **ESC** to pause or access the menu.

## Bahasa Indonesia
**Maze 3D** adalah game labirin bergaya retro yang terinspirasi dari game klasik seperti Wolfenstein 3D. Dibuat sepenuhnya dengan C# tanpa game engine eksternal, proyek ini menunjukkan kekuatan pemrograman murni.

### Fitur
*   **Mesin Raycasting 3D**: Dibangun dari nol menggunakan C# dan GDI+.
*   **Generasi Prosedural**: Setiap level memiliki tata letak labirin yang unik dan dijamin bisa diselesaikan.
*   **Visual**: Tekstur dinding, pintu, dan harta karun dibuat secara otomatis (tidak perlu download gambar).
*   **Tingkat Kesulitan Bertahap**: Level semakin sulit dengan lebih banyak harta karun.
*   **Tanpa Aset Eksternal**: Semua grafik dan suara dibuat oleh kode saat game berjalan!

### Cara Bermain
1.  Jalankan aplikasi.
2.  Gunakan **W, A, S, D** atau **Panah Keyboard** untuk bergerak dan menengok.
3.  Kumpulkan semua **Harta Karun** (Piala Emas) untuk mendapatkan poin.
4.  Temukan **Pintu KELUAR** (bertuliskan "EXIT") untuk lanjut ke level berikutnya.
5.  Tekan **ESC** untuk pause atau membuka menu.
